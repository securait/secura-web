# Secura IT

## 🔐 Soluciones de Seguridad y Monitoreo Empresarial

Protegemos, monitoreamos y aseguramos tu infraestructura tecnológica.

---

## 🚨 Problemas que resolvemos

- Ataques a aplicaciones web
- Falta de visibilidad de eventos de seguridad
- Accesos sin control a sistemas críticos
- Auditorías sin evidencia técnica

---

## 🛡️ Servicios

### 🔹 WAF (Web Application Firewall)
Protección contra ataques web (OWASP Top 10)

### 🔹 SIEM (Wazuh)
Monitoreo, correlación de eventos y alertas en tiempo real

### 🔹 PAM
Control y auditoría de accesos privilegiados

### 🔹 Hardening
Fortalecimiento de servidores bajo estándares (ISO 27001 / CIS)

### 🔹 Monitoreo
Visibilidad completa de tu infraestructura
